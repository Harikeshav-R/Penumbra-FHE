Penumbra
========

Penumbra is an open source homomorphic inference engine designed for privacy-preserving machine learning. It executes pre-trained models on encrypted data without ever decrypting the data or the intermediate activations.

.. admonition:: Warning: Pre-Alpha Software
   :class: warning

   Penumbra is pre-alpha cryptography software. Do not use it in production. It has not been audited.
